## Getting started

Requirements

- Node.js v14.18.2^
- NPM v6.14.0^

### Installation

```bash

git clone https://github.com/ShivangM/Kaimono-MongoDB-Atlas-Hackathon.git

cd frontend
npm install

cd backend
npm install
```

### NOTE

You'll need a `.env` file in the `backend` directory.

It looks like this -

```bash
PORT = 8080

MONGO_URI = Database connection String
```

### Starting the application

```bash
cd frontend

npm start
```

```bash
cd backend

node index.js
```
